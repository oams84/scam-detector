API Example

Use the API endpoint below to analyze messages programmatically.

Endpoint: /api/analyze

Method: POST

Content-Type: application/json

Sample request:

curl -X POST http://127.0.0.1:5000/api/analyze \ -H "Content-Type: application/json" \ -d '{"message":"Your bank account is suspended! Click here to verify: http://fake-bank.com"}'

Sample JSON body:

{ "message": "Congratulations! You won a gift card. Click here now!" }
